# FGANG Visuals

Free visual client mod for **Minecraft 1.21.4** on **Fabric Loader 0.17.3**.

Based on the open-source [executor-rblx/-visual](https://github.com/executor-rblx/-visual) Fabric base (CC0).

## Build

Requires JDK 21+.

```bash
./gradlew build
```

The built mod jar will be at `build/libs/fgangvisuals-<version>.jar`.

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.4.
2. Put `fgangvisuals-<version>.jar` into your `.minecraft/mods` folder.
3. Launch the game via the Fabric profile.

## Notes

- Authentication/HWID gating has been removed; the mod is public.
- Configs, friends, macros, and draggable HUD positions are stored under `.minecraft/fgangvisuals/`.
