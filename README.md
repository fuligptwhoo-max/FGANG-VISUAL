# FGANG Visuals

Визуальный клиентский мод для **Minecraft 1.21.1** на базе **NeoForge 21.1.224**.

Добавляет PVP-HUD, визуальные эффекты, оптимизации и кастомизацию интерфейса — всё настраивается через внутриигровое меню.

---

## 📋 Системные требования

| Параметр | Версия |
|----------|--------|
| Minecraft | 1.21.1 |
| NeoForge | 21.1.224+ |
| Java | 21+ |

---

## ⚡ Установка

1. Установите [NeoForge 21.1.224+](https://neoforged.net/) для Minecraft 1.21.1.
2. Скачайте последний `.jar` из раздела [Releases](../../releases).
3. Поместите файл в папку `.minecraft/mods/`.
4. Запустите игру через NeoForge-профиль.

---

## 🎮 Клавиши

| Клавиша | Действие |
|---------|----------|
| `Right Shift` | Открыть ClickGUI (настройка модулей) |
| `C` | Зум (приближение) |
| `Left Alt` | Freelook (свободная камера) |

> Клавиши можно изменить в стандартных настройках управления Minecraft (раздел "FGANG Visuals").

## 🛡️ PVP

### ArmorHUD
Отображает броню игрока с индикаторами прочности. Поддерживает вертикальный и горизонтальный режимы, процент и абсолютное значение прочности.

### KeyStrokes
Визуальное отображение нажатий клавиш WASD, пробела и кнопок мыши (LMB/RMB) прямо на экране. Цвет меняется при нажатии.

### Custom Crosshair
Полностью настраиваемый прицел:
- Свой цвет, прозрачность, размер, толщина
- Динамическое расширение при беге/стрейфе
- Точка в центре, обводка
- Возможность загрузить собственное изображение
- Поворот и смещение по X/Y

### Hit Color
При ударе по сущностям временно меняет их цвет (дефолт — красный). Помогает визуально фиксировать попадания.

### Toggle Sprint / Toggle Sneak
Фиксация бега/приседания по нажатию клавиши. Можно включить индикатор [SPRINTING] / [SNEAKING] на экране.

### CPS Counter
Счётчик кликов в секунду (CPS). Отображает левую/правую руку раздельно или суммарно.

### Coordinates
Координаты XYZ, направление (Facing) и название биома. Настраиваемый масштаб и позиция.

### FPS Counter
Счётчик FPS с цветовой индикацией (зелёный/жёлтый/красный в зависимости от значения).

### Ping Display
Задержка до сервера в миллисекундах. Цвет меняется в зависимости от пинга.

### TargetHUD
При наведении на игрока/моба показывает его имя, здоровье (полоска + цифры), броню и активные эффекты зелий.

### Reach Display
Показывает дистанцию до сущности в кроссхейре в момент удара. Полезно для тренировки варпа/реча.

### Combo Counter
Считает комбо (количество последовательных попаданий по одной цели без промаха или получения урона).

### InventoryHUD
Мини-версия хотбара и брони прямо на экране. Удобно для PVP-модов, где инвентарь не всегда виден.

### Time Display
Текущее реальное время. Поддерживает 12- и 24-часовой формат.

### Session Info
Статистика текущей сессии: убийства, смерти, K/D, время в игре.

### Memory Display
Отображает использование RAM (использовано / доступно в MB).

### Pot Counter
Счётчик зелий в хотбаре (Splash/Lingering Potion).

### W-Tap Indicator
При атаке по сущности показывает подсказку "W-TAP" — напоминание о технике отмены спринта для увеличения knockback.

## ✨ Visuals

### Hit Particles
При получении урона сущностью появляются цветные частицы. Можно настроить цвет, количество, размер и включить только при ударе игрока.

### Trail Effect
3D-трейл (след) за движением игрока. Настраиваемый цвет, ширина, длина и максимальное количество точек.

### Dynamic Lights
Факел, фонарь, лавовое ведро и другие светящиеся предметы в руках дают динамический свет вокруг игрока без установки шейдеров.

### Kill Effects
Визуальные эффекты при убийстве сущности:
- **Lightning** — частицы End Rod + звук грома
- **Explosion** — взрыв + огонь
- **Blood** — красные частицы (Dust)
- **Digital** — цифровые частицы с кастомным цветом

### FullBright
Максимальная гамма (яркость) без использования зелий или шейдеров. Работает даже в полной темноте.

### No Hurt Cam
Отключение тряски камеры при получении урона.

### Reduced Bobbing / View Bobbing
Настраиваемая сила покачивания камеры при ходьбе. Можно полностью отключить или уменьшить.

### Zoom
Плавное приближение камеры с настраиваемым FOV и скоростью. Работает как оптический зум.

### Freelook
Свободная камера при зажатии клавиши — можно осмотреться вокруг, не поворачивая персонажа.

### MotionBlur
Эффект смазывания при резких движениях камеры (экспериментальная реализация).

### No Overlay
Отключение визуальных наложений:
- Огонь на экране
- Тыква (при надевании)
- Эффект тошноты ( nausea )

### Low Fire
Уменьшает высоту отображения огня на экране, чтобы не закрывал обзор.

### Environment Editor
- **No Fog** — отключение тумана
- **No Weather** — отключение дождя/грозы
- **Clear Water** — прозрачная вода

### Screen Effects
- **Screen Shake** — тряска экрана при получении урона
- **Screen Flash** — вспышка при убийстве

### Hitbox & Chunk Borders
Визуализация хитбоксов сущностей и границ чанков (F3+G).

### Scoreboard Editor
Управление отображением табло.

### Chat Editor
Управление отображением чата.

## ⚙️ Optimizations

### Entity Culling
Не рендерит сущности за пределами указанной дистанции и за пределами поля зрения. Увеличивает FPS в локациях с большим количеством мобов/игроков.

### Particle Optimizer
Автоматически ограничивает количество частиц при просадках FPS. Переключает настройку частиц с "All" на "Decreased" при низком FPS.

### Render Optimizer
Автоматически переключает графику на "Fast" при просадках ниже 30 FPS (если включена опция Fast Render).

### FPS Smooth
Сглаживание показаний FPS (Moving Average) для более стабильного отображения.

### BlockEntity Optimizer
Не рендерит блок-entities (сундуки, печки, знаки и т.д.) на большой дистанции.

### Entity Optimizer
Прерывает обновление (tick) дальних сущностей с интервалом в 1 секунду вместо каждого тика.

### Memory Optimizer
Периодический вызов `System.gc()` для принудительной очистки памяти.

## 🖥️ GUI

### ClickGUI (`Right Shift`)
Внутриигровое меню настройки всех модулей с анимациями. Разделены категории: PVP, Visuals, Optimize, Crosshair, GUI.

### HUD Editor
Режим редактирования позиций HUD-элементов на экране.

### ModuleList
Список активных модулей с радужной анимацией и настраиваемым расположением.

### Watermark
Кастомный текст (по умолчанию название мода) с настраиваемым масштабом, цветом и позицией.

### Notifications
Всплывающие уведомления при включении/выключении модулей.

## 🛠️ Конфигурация

Все настройки сохраняются в:
```
.minecraft/config/fgangvisuals-client.toml
```

Через ClickGUI можно изменить:
- Позицию, масштаб и цвет каждого HUD-элемента
- Включение/отключение модулей
- Значения FOV, дистанций, интервалов
- Цвета и стили прицела
